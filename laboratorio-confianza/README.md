# Laboratorio de Confianza

Aplicación web desarrollada en **PHP + MariaDB**, integrada con **Joomla** como interfaz de presentación, que permite realizar un test completo de confianza de 40 ítems con cálculo automático por bloques, perfiles interpretativos y almacenamiento de resultados.

## Funcionalidades

- Test de 40 preguntas
- Escala Likert de 1 a 5
- Preguntas directas e inversas
- Cálculo automático por bloques
- Perfil principal interpretativo
- Guardado de resultados en MariaDB
- Página de visualización de resultados guardados
- Integración del test con Joomla
